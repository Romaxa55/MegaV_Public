## Возможности

- **Сквозное шифрование** — защита данных в любой сети
- **Без логов** — мы не собираем и не храним данные о вашем трафике
- **Мультиплатформенность** — один аккаунт на Android, iOS, macOS и Windows
- **Блокировка рекламы** — встроенная защита от трекеров и рекламы
- **Раздельное туннелирование** — выбирайте, какие приложения используют VPN
- **Kill Switch** — автоматическая защита соединения
- **15 языков** — полная локализация
- **6 тем** — динамическая система тем
