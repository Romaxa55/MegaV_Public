- **Открытый код клиента** — код приложения доступен для аудита. Никаких скрытых бэкдоров.
- **Без аккаунта** — пользуйтесь бесплатными серверами без регистрации и без указания email.
- **Защита от утечек DNS** — все DNS-запросы разрешаются внутри зашифрованного туннеля.
- **Защита от утечек IPv6** — IPv6 отключается внутри туннеля, чтобы исключить раскрытие личности.
