## Privacy & Security

- **Zero-log architecture** — We do not store connection timestamps, IP addresses, DNS queries, or any traffic data.
- **RAM-only servers** — Server memory is wiped on every reboot. No data survives a power cycle.
- **Open-source client** — The app code is publicly auditable. No hidden backdoors.
- **No account required** — Use the free V2Ray servers without creating an account or providing an email.
- **DNS leak protection** — All DNS queries are resolved inside the encrypted tunnel.
- **IPv6 leak protection** — IPv6 is disabled within the tunnel to prevent identity exposure.
